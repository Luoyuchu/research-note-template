# Resources

这里不是正式知识库，而是原始材料和低加工参考的存放区。

适合放：

- AI 原始对话
- 临时摘录
- 尚未提炼的网页或说明文档
- 以后可能会被整理进项目页、知识页或写作页的来源材料

不建议直接放：

- 已经稳定的工具方法
- 已经整理好的项目结论
- 已经进入成稿阶段的写作内容

## Start Here

- AI 原始材料示例：[[04_Resources/AI_Raw/README|AI_Raw]]

## Common Moves

- 想保留长原文：放这里
- 想保留当天真正有用的几点：摘到 `01_Daily_Logs/`
- 想变成稳定知识或项目结论：再提炼出去

## How This Layer Connects To Others

- `Resources -> Daily`
  - 在当日日志里只摘录你当天真正用到的几点
- `Resources -> Project`
  - 对项目推进有直接作用的结论，提炼后进入项目页
- `Resources -> Knowledge`
  - 跨项目可复用的方法和概念，提炼后进入知识层
- `Resources -> Writing`
  - 当来源材料开始服务于某个交付物时，在写作页中重组表达

推荐流转：

`Resources -> Daily / Project -> Knowledge / Writing`
