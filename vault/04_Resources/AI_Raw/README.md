# AI Raw

这里放原始 AI 对话和长讨论记录。

保留它们的原因不是为了直接反复阅读全文，而是为了：

- 保留来源
- 保留上下文
- 方便以后回溯某个判断从哪里来

## Best Practice

- 原文尽量完整保留
- 正式知识页里不要直接塞大段原文
- 真正长期有用的内容，应该写成摘要或结论页，再链接回这里

## Recommended Pattern

1. 原始对话保留在这里
2. 在 `01_Daily_Logs/` 中摘录当天真正有用的几点
3. 再把稳定内容提炼到 `02_Projects/`、`03_Knowledge/` 或 `05_Writing/`
